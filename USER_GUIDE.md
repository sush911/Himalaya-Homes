### For Buyers/Renters

#### Viewing Ratings on Property List
- When browsing properties, you'll see a **star rating** (1-5) below the property location
- A **review count** shows how many people have reviewed the property
- Properties with more stars have been rated highly by other users

#### Reading Reviews on Property Detail Page
1. Navigate to any property
2. Scroll down to the **"Reviews"** section
3. You'll see:
   - **Average Rating** with visual stars (top-right)
   - **All User Reviews** showing:
     - User name
     - Star rating
     - Comment (if provided)
     - Date of review

#### Leaving a Review
1. **Login first** (required to leave reviews)
2. Go to property detail page
3. In the **Reviews** section, find the review form at the top
4. **Rate the property**: Click on stars (1-5) to set your rating
5. **Optional**: Add a comment (up to 500 characters)
6. Click **"Submit Review"** button
7. Your review will appear immediately in the reviews list
8. **Update**: If you review the same property again, your previous review will be updated

### For Admin/Sellers

#### Marking Properties as Verified
1. Go to Admin Panel
2. Navigate to **Admin Properties** section
3. Find the property in the list
4. Click **"Verify"** button (if available)
5. The property will now show a **green "Verified" badge**
6. Verified properties appear more trustworthy to buyers

#### Viewing Property Reviews
- In Admin Panel â†’ Admin Properties, you can see review counts
- Click on a property to see all reviews in the detail view
